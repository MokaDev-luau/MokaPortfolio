Rocket Launcher Integration

I scripted a fully working rocket launcher for a Roblox game, complete with custom animations. The project involved creating a weapon system with multiple meshes and ensuring proper player interaction for usage.

Script for Gun Position Adjustment:

        
local gun = script.Parent
local player = game.Players.LocalPlayer
local character = player.Character or player.CharacterAdded:Wait()
local humanoid = character:WaitForChild("Humanoid")
local gunAttachment = gun:WaitForChild("Attachment")

local equipped = false

local function adjustGunPosition()
    if equipped then
        gun.CFrame = character.RightHand.CFrame * CFrame.new(0, 0, -0.5)
    end
end

gun.Equipped:Connect(function()
    equipped = true
    adjustGunPosition()
end)

gun.Unequipped:Connect(function()
    equipped = false
end)

humanoid.Running:Connect(function(speed)
    if equipped then
        adjustGunPosition()
    end
end)

NPC Combat System

Developed an NPC that chases and fights the player, including health bars, animations, and combat mechanics. This NPC is integrated into the game to trigger at certain locations for dynamic gameplay.

Script for NPC Combat:

        
local npc = script.Parent
local player = game.Players.LocalPlayer
local target = player.Character:WaitForChild("HumanoidRootPart")
local npcHumanoid = npc:WaitForChild("Humanoid")
local attackRange = 10

local function moveTowardsTarget()
    while npcHumanoid.Health > 0 do
        if (npc.Position - target.Position).Magnitude <= attackRange then
            npcHumanoid:MoveTo(target.Position)
        else
            npcHumanoid:MoveTo(npc.Position)
        end
        wait(1)
    end
end

moveTowardsTarget()
